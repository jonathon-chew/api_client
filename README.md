# API Client CLI

## Summary
Build a command-line HTTP client for sending requests and inspecting responses, similar to a minimal Postman for the terminal.

## What You'll Learn
- HTTP methods and headers
- request body formatting
- response parsing
- CLI ergonomics

## MVP
- support GET, POST, PUT, and DELETE
- send headers and JSON bodies
- print status, headers, and response body
- save named requests for reuse

## Stretch Goals
- auth helpers
- file uploads
- collections of requests
- response timing metrics
